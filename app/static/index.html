Import Contacts

Upload an Excel (.xlsx) or CSV file — the AI will automatically call each number

📥 Upload Contact File
📁
Drag & drop or click to browse
Supports: .xlsx, .xls, .csv — Columns: name, phone
📌 File Format Guide

Your Excel or CSV file needs at least a phone column. A name column is optional but recommended.

✅ Accepted column names:
name, full_name, Full Name
phone, Phone, mobile, Mobile
mobile_number, phone_number
contact, number, cell, tel
📋 Example CSV:
name,phone
Ravi Kumar,+919876543210
Priya Sharma,+919123456789
Amit Singh,9001234567
📞 After importing, go to Campaigns and click Call All to trigger real outbound calls.
🔑 Twilio credentials must be set as env vars: TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_FROM_NUMBER, TWILIO_WEBHOOK_BASE_URL
